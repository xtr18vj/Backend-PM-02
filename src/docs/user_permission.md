# User Field-Level Permissions

| Field             | Editable by User | Editable by Admin |
|------------------|------------------|-------------------|
| Name             | ✅ Yes           | ✅ Yes            |
| Email            | ❌ No            | ✅ Yes            |
| Role             | ❌ No            | ✅ Yes            |
| Status           | ❌ No            | ✅ Yes            |
| Profile Photo    | ✅ Yes           | ✅ Yes            |
